# markdown-formatter

基于[markdown-formatter v0.9.7](https://github.com/sumnow/markdown-formatter)。

## 出发点

由于我主要用来写个人 `wiki` ，`markdown-formatter` 的很多操作与 `docsify` 不兼容，因此基于此魔改一套适用于 `docsify` 的。

## roadmap

* [x] 更改原版的 `code area` 语言，从大写改为小写。
* [x] 优化标题换行过多。
* [x] 支持 `docsify` 高亮块快捷生成。
* [x] 优化行内代码块首尾空格。
